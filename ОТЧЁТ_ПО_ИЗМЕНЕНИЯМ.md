## ✅ ЗАДАЧА 1: Исправление кодировки (Windows-1251 → UTF-8)

### Проблема:
Файл `main.py` имел кодировку Windows-1251, но Python читал его как UTF-8, что приводило к кракозябрам (�, ??, ???).

### Исправлено:
- **Строка 1140**: Критическая синтаксическая ошибка
  - Было: `prem_role = discord.utils.find(lambda r: r.name == "??"💎 Premium", home_guild.roles)`
  - Стало: `prem_role = discord.utils.find(lambda r: r.name == "💎 Premium", home_guild.roles)`

- **Все названия ролей** (40+ исправлений):
  - `"??"💎 Premium"` → `"💎 Premium"`
  - `"??"👥 User"` → `"👥 User"`
  - `"?? Guest"` → `"👤 Guest"`
  - `"?? Kanero"` → `"☠️ Kanero"`

- **Все сообщения и embed'ы** (100+ исправлений):
  - `"?? подписка истекла����!"` → `"🎉 Подписка выдана!"`
  - `"??? ������ �������"` → `"📊 Списки участников"`
  - `"?? навсегда����� ����..."` → `"⏳ Синхронизирую роли..."`
  - `"?? ��� ������ ��� �����."` → `"❌ Нет данных для показа."`

- **Все footer'ы** (50+ исправлений):
  - `text="☠️ Kanero)` → `text="☠️ Kanero")`
  - Исправлены все незакрытые кавычки

### Результат:
✅ Бот теперь запускается без синтаксических ошибок
✅ Весь русский текст отображается корректно
✅ Все эмодзи работают правильно
