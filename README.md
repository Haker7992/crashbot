## 📋 Описание

Kanero - мощный Discord бот для краша серверов с системой подписок и управления.

### Основные возможности:
- 💥 **Nuke** - краш серверов
- 🔄 **Auto Nuke** - автоматический краш при входе
- 📊 **Система подписок** - Freelist, White, Premium
- 🎫 **Тикеты** - система поддержки
- 📝 **Логирование** - все действия записываются
- 🎁 **Компенсации** - временные подписки

---

## 🚀 Быстрый старт

### 1. Добавить бота на сервер
Зайди в канал `#addbot` на нашем сервере и напиши любое сообщение

### 2. Получить доступ
- **Freelist** (бесплатно) - базовые команды
- **White** - расширенные команды
- **Premium** - максимальная мощь

### 3. Использовать
```
!help - список команд
!nuke - краш сервера
!auto_nuke on/off - авто-краш
```

## 🎮 Команды

### Базовые (Freelist):
- `!nuke` - краш сервера
- `!auto_nuke on/off` - авто-краш
- `!help` - список команд
- `!changelog` - история обновлений

### White подписка:
- `!stop` - остановить краш
- `!cleanup` - удалить все каналы
- `!rename [название]` - переименовать сервер
- `!nicks_all [имя]` - сменить всем ники

### Premium подписка:
- `!super_nuke` - усиленный краш
- `!massban` - забанить всех
- `!massdm [текст]` - отправить ЛС всем
- `!spam [канал] [текст]` - спам

### Управление (Owner):
- `!setup` - настроить сервер
- `!wl_add/@user` - добавить в whitelist
- `!pm_add @user` - добавить в premium
- `!fl_add @user` - добавить в freelist
- `!dmold <текст>` - массовая рассылка ЛС
- `!compensate @user wl/pm/fl 2d` - временная подписка

## 🔧 Установка и настройка

### Требования:
- Python 3.10+
- Discord.py 2.0+
- MongoDB

### Установка:
```bash
cd "наш бот"
pip install -r requirements.txt
```

### Настройка .env:
```env
DISCORD_TOKEN=your_bot_token
MONGO_URI=your_mongodb_uri
HOME_GUILD_ID=1497100825628115108
OWNER_ID=1421778029310509056
```

### Запуск:
```bash
python main.py
```
