## 🎯 Возможности

### 💀 Краш серверов
- **!nuke** — базовый краш (удаление каналов, ролей, спам)
- **!super_nuke** — усиленный краш с банами (Premium)
- **!owner_nuke** — полный краш без ограничений (Owner)
- **!cleanup** — очистка после краша

### 🔐 Система подписок
- **📋 Freelist** — бесплатный доступ (через #addbot)
- **✅ Whitelist** — базовая подписка (кастомный текст)
- **💎 Premium** — расширенная подписка (все функции)
- **⏰ Временные подписки** — компенсации с автоистечением

### 🛡️ Защита
- Блокировка деструктивных команд на домашнем сервере
- Защита от самоуничтожения
- Система прав доступа
- Логирование всех действий

### 🤖 Автоматизация
- **!auto_nuke** — автокраш при входе бота
- **!auto_super_nuke** — автокраш с банами (Premium)
- **!auto_owner_nuke** — полный автокраш (Owner)

### 🎫 Дополнительно
- Система тикетов с кнопками
- Автоматическая настройка сервера (!setup)
- Статистика в голосовых каналах
- Массовые операции (бан, рассылка, спам)

## 🚀 Быстрый старт

### 1. Установка

```bash
# Клонировать репозиторий
git clone https://github.com/yourusername/kanero-bot.git
cd kanero-bot

# Установить зависимости
pip install -r requirements.txt
```

### 2. Настройка

Откройте `config.py` и укажите:

```python
TOKEN = "ваш_токен_бота"
OWNER_ID = ваш_discord_id
```

### 3. Запуск

```bash
python main.py
```

Или используйте `start.bat` (Windows)

## 🎮 Основные команды

### Для всех (Freelist+):
```bash
!nuke              # Базовый краш сервера
!cleanup           # Очистка после краша
!auto_nuke on/off  # Автокраш при входе
!help              # Список команд
!info              # Информация о подписке
```

### Premium:
```bash
!super_nuke        # Усиленный краш с банами
!massban           # Массовый бан участников
!massdm <текст>    # Массовая рассылка
!spam <N> <текст>  # Спам в канале
```

### Owner:
```bash
!wl_add @user      # Добавить в whitelist
!pm_add @user      # Добавить premium
!компенсация pm 1d # Выдать временную подписку
!setup             # Настроить сервер
```

[Полный список команд →](ИНСТРУКЦИЯ.md#команды-бота)

## 🔐 Система прав

| Уровень | Получение | Команды | Роль |
|---------|-----------|---------|------|
| **Freelist** | Бесплатно (#addbot) | !nuke, !cleanup | 👥 User |
| **Whitelist** | Платно | + кастомный текст | ✅ White |
| **Premium** | Платно | + super_nuke, массовые | 💎 Premium |
| **Owner** | Создатель | Все команды | 👑 Owner |

## ⏰ Временные подписки

### Команда компенсации:
```bash
!компенсация pm 1d    # Premium на 1 день
!компенсация wl 7d    # Whitelist на 7 дней
!компенсация fl 12h   # Freelist на 12 часов
```

### Как работает:
1. Овнер отправляет команду
2. Бот создаёт сообщение с кнопкой
3. Пользователи нажимают и получают подписку
4. Подписка автоматически истекает
