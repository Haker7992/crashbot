## v2.6 - "Система управления листами" (29.04.2026)

### ✨ Новое:
- **Роли**:
  - 🎬 Media - для медиа контента
  - 🤝 Friend - для друзей
  
- **Каналы**:
  - 🎬・медиа - медиа контент с slowmode 5 минут
  - 📋・выдача-листа - управление подписками (только Owner)
  - 🎁・компенсация - временные подписки
  
- **Категория ТИКЕТЫ**:
  - Создаётся над категорией АДМИН
  - Все тикеты перемещаются туда
  
- **Команда !dmold**:
  - Массовая рассылка ЛС всем кому бот писал в #addbot
  - Только для Owner
  - С подтверждением и прогрессом

### 🔧 Улучшения:
- Команды управления подписками работают только в #выдача-листа
- Команда !list доступна везде
- Постоянное сообщение в #addbot с инструкцией
- Переименован канал "выдача-листа" → "лог-листа" в категории АДМИН
- Улучшены права доступа к категориям АДМИН и TESTS
- Пинг пользователя в приветственном сообщении

### 🐛 Исправления:
- Исправлены все кракозябры в сообщениях
- Исправлена ошибка создания тикетов
- Исправлен текст "Твоя помощь не заставится" → "Мы всегда поможем"
- Исправлена ошибка CommandNotFound
- Исправлена блокировка команд на домашнем сервере
- Исправлена ошибка "Unknown Channel" при выдаче Guest роли
- Улучшено удаление ролей и каналов в !setup
